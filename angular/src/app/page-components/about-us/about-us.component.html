About Movie App

Create a truly engaging and modern website or application that attracts visitors and encourages them to share it with their friends. The API is excellently suited for building high-traffic, profitable websites or applications, as well as for personal projects. The API contains two types of endpoints. The “top 100 movies list” endpoint responds to the request with a complete list of 100 movies, including their images, titles, ratings, and IDs. The received ID can then be used to retrieve detailed information and the trailer for a specific movie. The ID also represents the movie’s position on the list. For example, “The Shawshank Redemption,” occupying the first position on the list, has a ranking of 1 and an ID of “top1.” To access this movie, the ID needs to be added after the “top 100 movies list” link.
